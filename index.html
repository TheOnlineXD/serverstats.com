Bot Features

Fancy counter

This bot supports multiple counters that automatically updates, the counter that exists now are all members, members, bots, channels, categories, voice, text, roles, online, offline, role, emojis, static, animated, bans, connected, status, playing and minecraft.

Easy to setup

Some bots are a pain to set up but not this bot but, just type s/setup or @ServerStats setup and the bot will run the setup (this includes creating the channels and updating the counters).

Fully customizable

You can change the counter in every single way, you don't like the text or you don't want them under a category? no problem. it's also easy to change because most thinks can be done like normal (like deleting a channel to disable it).

Show info

The bot can show you server, user, role, channel and invite information, all information command show advanced information in a nice looking embed.

24/7 Uptime

The bot is built to run 24/7 (it's really stable and reliable) and problems are fixed really fast, the only downtime will be for bot/server update or discord problems.

MongoDB database

The bot uses a MongoDB database that is hosted on the same server without connections to the outside world, This make it really has and fast (to see your data use s/data <type>).
